OpenSearch Troubleshooting Guide

Comprehensive solutions for common OpenSearch challenges

Important Note: This guide and the associated chatbot provide troubleshooting guidelines only. They do not automatically fix cluster issues. All recommended actions require manual verification and implementation by a qualified administrator.

Alert Status Guide

Yellow Cluster Status

Common Causes:

  • Unassigned replica shards
  • Node capacity issues
  • Network connectivity problems

Solution Steps:

  1. Check unassigned shards
  2. Verify node capacity
  3. Review cluster settings
  4. Monitor disk space

Red Cluster Status

Common Causes:

  • Primary shard failures
  • Node failures
  • Disk space issues

Solution Steps:

  1. Check node availability
  2. Verify master node status
  3. Review error logs
  4. Check disk space

Essential Commands

Cluster Health Check

GET _cluster/health

Shows overall cluster status and shard information

Index Status

GET _cat/indices

Lists all indices with their status and basic info

Shard Allocation

GET _cat/shards

Shows detailed shard allocation across nodes
