Components Used:

  • Amazon Lex: Chatbot service for natural language processing
  • AWS Lambda: Serverless compute for bot logic
  • OpenSearch: Search and analytics engine
  • AWS Amplify: Hosting for this documentation

Flow Description:

  1. User interacts with the Lex chatbot
  2. Lex triggers Lambda function
  3. Lambda processes the request and queries OpenSearch if needed
  4. Response includes links to this documentation
